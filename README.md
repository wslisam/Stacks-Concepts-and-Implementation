Welcome to the **Stack Learning Tool**! This web application is designed to help students easily understand one of the most fundamental data structures in computer science: **The Stack**. 

Whether you are preparing for your HKDSE ICT exams or just starting your journey in programming, this tool will guide you through the concepts visually and interactively.

## 🎯 Learning Objectives
By using this tool, you will be able to:
1. Define a Stack as a **Last-In-First-Out (LIFO)** data structure.
2. Understand how to implement a Stack using an array and a `top` pointer variable.
3. Construct algorithms for **Push** and **Pop** operations, and understand how to handle errors like **Stack Overflow** and **Stack Underflow**.
4. Trace the state of a stack given a sequence of operations.

## 🚀 How to Use This App

### 1. Read the Concepts & Guiding Prompts
Start by reading the introductory sections. Pay special attention to the colored **Guiding Prompts** (Think About It, What happens when we Pop?, HKDSE Exam Trap). These boxes answer common questions and highlight important exam tips!

### 2. Play with the Interactive Simulator
The best way to learn is by doing! Scroll down to the **Interactive Stack Simulator**:
*   **Push:** Type a value (or leave it blank for a random number) and click "Push". Watch how the `TOP` pointer moves *before* the item is added.
*   **Pop:** Click "Pop" to remove an item. Notice that the item doesn't disappear immediately—it becomes a "ghost" (logically deleted) and the `TOP` pointer moves down.
*   **Test the Limits:** Keep pushing until the stack is full to trigger a **Stack Overflow** error. Keep popping until it's empty to see a **Stack Underflow** error.
*   **Trace Log:** Look at the terminal window on the left to see a step-by-step history of your actions.

### 3. Review the Code
Once you understand the visual concept, look at the **Code Implementation** section. You can toggle between **Python** and **C++**. Try to match the 3 steps in the code comments (Check limits -> Move pointer -> Insert/Retrieve) with what you saw in the simulator.

### 4. Test Your Knowledge
Finish your learning session by taking the short **Quiz** at the bottom of the page. It will test your understanding of the `top` pointer, array indices, and the LIFO principle.

## 🌐 Language Support
You can switch between **English** and **中文** at any time by clicking the language button in the top right corner. The Chinese terminology is tailored for Hong Kong students (e.g., 堆疊, 陣列, 溢位).
